Principais formatações

Negrito

Fazemos um TEXTO EM NEGRITO usando a tag <b> (não semântica)

Para fazer um TEXTO EM NEGRITO de forma semântica usamos a tag <strong>

ITÁLICO OU ÊNFASE

Para colocar um frase em itálico usamos a tag <i> (não semântica)

Para colocarmos uma frase em destaque usamos <EM>(semântica)

TEXTO MARCADO

Fazemos um texto marcado usando a tag <MARK>

Aqui temos outro exemplo de marcação de texto.

TEXTO GRANDE E PEQUENO

Criamos um texto pequeno usando a tag <small>

Criamos um texto grande usando a tag <big> (obsoleto)

TEXTO DELETADO

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

TEXTO INSERIDO

Podemos indicar um texto inserido para dar uma ênfase e indicar que ele foi adicionado depois.

TEXTO SOBRESCRITO

Para inserir coisas tipo X20+3. TAG: <sup>

TEXTO SUBSCRITO

Para inserir coisas tipo H2O. TAG: <sub>
